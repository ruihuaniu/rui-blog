### ‚õ≥  Overview

Binary Tree is an important data structure, and it's widely used in BFS, DFS, DP, Back tracking, Divide and conquer algorithms.

### üìÉ  Analysis
:::tip
 3 Tree Traversals
 - Inorder: Left, Root, Right
 - Preorder: Root, Left, Right
 - Postorder: Left, Right,Root
:::

**Examples:**

![Binary tree](/img/binary-tree.webp)
```
Inorder: 4 2 5 6 3 7 1
Preorder: 1 2 4 5 3 6 7
PostOrder: 4 5 2 6 7 3 1

```

const res = {}
function traverse(node, depth) {
  if(node===null) return;
  //console.log("node", node, "depth",depth);
  res[depth] = {node, depth};
  traverse(node.left, depth+1);
  traverse(node.right, depth+1);
