### ‚õ≥  Overview

Binary Tree is an important data structure, and it's widely used in BFS, DFS, DP, Back tracking, Divide and conquer algorithms.

### üìÉ  Analysis
:::tip
 3 Tree Traversals
 - Inorder: Left, Root, Right
 - Preorder: Root, Left, Right
 - Postorder: Left, Right,Root
:::

**Examples:**

![Binary tree](/img/binary-tree.webp)

**2. Print the number of leafs of each node**

```jsx live noInline
const res = {}

function traverse(node, depth) {
  if(node===null) return 0; 
  const leftCount= traverse(node.left, depth+1);
  const rightCount=traverse(node.right, depth+1);
  res[depth] = {node, count:leftCount+rightCount+1};
   // console.log("node", node, "count",leftCount+rightCount+1);
  return leftCount+rightCount+1;
};

const testCases=[binaryTree]

render(
   <TestRun
    codeFuc= {traverse}
    testCases={testCases}
    res={res}
  />
)
```
